What is sub > txt ?

This tool allows you to convert the popular softsub format .srt to plain text.
For specific reasons the .ass file format is also accepted.

This tool is useful for searching for keywords in subtitle files.
The timestamps of each line has been made visible to make this easier.
Searching can be done by pressing CTRL+F on the keyboard.

The user is able to keep adding new subtitle files; the texts will continue being added to the page. They can easily be navigated through with buttons.

More supported file formats (including .zip) might be added in the future.

This tool was made by Jeroen Bol from the Netherlands.

Subtitle files can be found on the following websites:

opensubtitles.org downsub.com
addic7ed.com podnapisi.net
